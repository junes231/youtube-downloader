# YouTube Video Downloader Web Page

A simple and responsive web tool to extract YouTube video ID and redirect to the download page (via ssyoutube.com). No installation needed â€” works on both mobile and desktop.

## Features

- Automatically extracts video ID from `youtube.com` and `youtu.be` links
- Redirects to [ssyoutube.com](https://ssyoutube.com) for download
- Mobile-friendly responsive design
- 100% frontend (HTML + JS), no backend or server needed

## How to Use

1. Upload the `index.html` file to your GitHub repository.
2. Go to **Settings > Pages**.
3. Set the source to the `main` branch and root (`/`) directory.
4. After deployment, your site will be available at: https://junes231.github.io/youtube-downloader/
